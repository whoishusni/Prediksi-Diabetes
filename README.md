This is a program that I created to detect / classify whether a person has diabetes or not by using the SVC (Support Vector Machine Classifier) ​​Model using several parameters such as:
1.   Glucose
2.   Blood Pressure
3.   Skin Thickness
4.   Insulin
5.   BMI
6.   Age
